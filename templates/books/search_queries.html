🏪 Запрос 2: Книги, продающиеся в магазинах Москвы

ORM Запрос: Book.objects.filter(stores__city='Москва').select_related('author', 'publisher').prefetch_related('genre', 'stores').distinct()
✅ Оптимизация: prefetch_related() для stores (ManyToMany)
{% for book in books_in_moscow %}
{{ book.title }}
Автор: {{ book.author.name }}
{% for genre in book.genre.all %} {{ genre.name }} {% endfor %} 📍 Магазины: {% for store in book.stores.all|slice:":3" %} {{ store.name }}{% if not forloop.last %}, {% endif %} {% endfor %}
{% empty %}
Книги не найдены
{% endfor %}

⭐ Запрос 3: Книги со средней оценкой выше 4.5

ORM Запрос: Book.objects.annotate(avg_rating=Avg('reviews__rating')).filter(avg_rating__gt=4.5).select_related('author', 'publisher').prefetch_related('reviews')
✅ Оптимизация: prefetch_related() для reviews + агрегация Avg
{% for book in high_rated_books %}
{{ book.title }} {% with avg=book.reviews.all|dictsort:"rating" %} ⭐ {{ book.reviews.all|length }} отзывов {% endwith %}
Автор: {{ book.author.name }}
{% for review in book.reviews.all|slice:":2" %} 💬 "{{ review.comment|truncatechars:50 }}" ({{ review.rating }}/5)
{% endfor %}
{% empty %}
Книги со средним рейтингом выше 4.5 не найдены
{% endfor %}

📊 Запрос 4: Количество книг в каждом магазине

ORM Запрос: Store.objects.annotate(book_count=Count('books')).order_by('-book_count')
✅ Агрегация Count и сортировка
{% for store in stores_with_book_count %}
{{ store.name }} {{ store.book_count }} книг
{% empty %}

{% endfor %}

🕒 Запрос 5: Магазины с книгами после 2010 года (сортировка по количеству книг)

ORM Запрос: Store.objects.filter(books__published_date__year__gt=2010).annotate(book_count=Count('books')).distinct().order_by('-book_count')
✅ Фильтрация по дате + агрегация + distinct()
{% for store in stores_with_recent_books %}
{{ store.name }} ({{ store.city }}) 📖 {{ store.book_count }} книг после 2010
{% empty %}
Магазины не найдены
{% endfor %}

🚀 Оптимизация запросов

✅ select_related() - использован для ForeignKey (author, publisher) - сокращает количество запросов с N+1 до 1
✅ prefetch_related() - использован для ManyToMany (genre, stores) и обратных связей (reviews)
✅ annotate() + Count/Avg - для агрегации данных на уровне БД
✅ distinct() - для устранения дублей при ManyToMany фильтрации
